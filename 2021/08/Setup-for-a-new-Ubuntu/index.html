APT

Explaination

  • sed = Stream EDitor
  • -i = in-place (i.e. save back to the original file)
  • The command string
    • s = the substitute command
    • cn.archive.ubuntu.com = a regular expression describing the word to replace (or just the word itself), it depends, can be cn.archive.ubuntu.com or archive.ubuntu.com, depends on your OS
    • mirrors.aliyun.com = I prefer Aliyun mirror, also you may choose USTC or Tshinghua
    • g = global (i.e. replace all and not just the first occurrence)
  • /etc/apt/sources.list = the file name
