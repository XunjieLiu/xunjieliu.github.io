How to install Ubuntu on a new machine

A guide to help you install Ubuntu

Requirements

  • At least 25 GB free space for normal installation, or 8.6 GB for minimal installation
  • 2 GHz dual core processor
  • 4 GiB RAM (system memory)
  • Remember to back your file on the installation disk
  • Network, WiFi or Cable

Boot from external drive

del, F2, F12, esc into BIOS panel, shortcuts are different according to different motherboard

Choose your USB drive to boot

Installation

After booting from USB drive, choose Install Ubuntu, words like that

Then you shall see this

  1. Choose minimal installation, to save time
  2. Do not choose Download updates while installing Ubuntu, this option would download and upgrade packages, which is slow

Click Continue:

Choose Something else, for custom disk allocation, you shall see something like this:

Disk info could be different in this interface, according to your conditions

  1. Choose which disk you wanna place this Ubuntu
  2. free space referers those space, has a partition table, but is not allocated to partitions
  3. If you cannot see free space under /dev/sda, click on the dev/sda first, then click on this button
  4. Once you have a free space, you can create new partitions in it, use plus button here to create, select specific partition and click minus button to delete

Something you need to pay attention:

  1. Operations would not be executed, unless you press the Install Now button, so don’t be panic
  2. If your disk is not brand new, has something in it, remember to back it up before the installation
  3. If your disk is brand new, you need to use New Partition Table button to create free space
  4. If your disk is not clean, click these partitions you wanna delete, then use button in 4 to delete it
  5. Then use button in 4 to create new partitions
  6. Normally, I would suggest 1 partition within one disk, and it is mounted in /
  7. Windows file system format is NTFS, for Linux, Ext4 is more common

When you click plus button in place 4:

  1. The first Size, just keep it to default, because default would use every space for this partition, and that’s what I want
  2. Choose Primary for the type of the new partition
  3. Do nothing to the next option
  4. Choose Ext4 as the file system format for the Ubuntu
  5. Mount point to /, means that every system files are under /
  6. Click OK, do be panic, nothing is executed yet

If you are pretty sure what you are doing, click Install Now

Make sure you know WHAT YOU ARE DOING

Then select your current location, normally its Shanghai

Some configuration

Something I need to explain before you continue:

  1. Your name: useless normally, displayed when you login, then useless, in most time
  2. Your computer's name: important, displayed when you communicate to others internet, like SSH, Hostname etc.
  3. Pick a username: YOUR NAME, a username for you
  4. Choose a password: password

Then continue, basically, its done
